> [!IMPORTANT]
> **Zero-Trust y 100% de Privacidad.** La aplicación es un único archivo HTML monolítico. Todos los cálculos centrales, la generación de enlaces y el análisis se realizan **estrictamente de forma local en la memoria RAM de tu dispositivo**. Las claves API de IA se guardan solo en tu caché local y se envían directamente a los proveedores (sin servidores intermediarios).

> [!TIP]
> **Preparado para PWA y Móviles:** Abre el sitio en un smartphone (iOS/Android) y toca "Añadir a la pantalla de inicio". ¡La aplicación se instalará como un programa completamente nativo que funciona incluso sin conexión (incluyendo IA Offline)!

### 🔥 Características Principales (¿Qué hay dentro?)

* 🌌 **Omni-AI Bridge (IA Online):** ¡Conecta tus claves API y el chatbot se convierte en un genio! Soporte directo integrado para `DeepSeek v4`, `ChatGPT (OpenAI)`, `Claude 3.5 (Anthropic)`, `Gemini` y modelos gratuitos `Qwen/Llama 3` (a través de OpenRouter). La red neuronal lee tu código y ayuda a optimizarlo.
* 🤖 **IA Offline con Aprendizaje Automático:** ¿No tienes internet o claves? ¡No hay problema! El bot offline integrado encontrará errores de sintaxis (comas sueltas, comillas simples) y los **corregirá automáticamente** con el comando "fix". Además, puedes *enseñarle* nuevos comandos con: `learn: X = Y`.
* 📂 **Arrastrar y Soltar Global:** Simplemente arrastra y suelta cualquier archivo `.json`, `.yaml` o `.txt` desde tu escritorio directamente a la ventana del navegador para un análisis instantáneo, o suéltalo en el chat de IA para un análisis neuronal detallado.
* 🎨 **Interfaz Game Engine (Emulador y Redimensionable):** Libertad absoluta de interfaz. Arrastra las ventanas en cualquier dirección para cambiar su tamaño. El **Emulador Móvil** integrado permite probar diseños móviles directamente en tu PC. ¡Personaliza los colores y comparte tu tema en 1 clic!
* 🎓 **Academia Interactiva:** Curso integrado de 15 pasos: desde tipos básicos de JSON hasta configuraciones complejas de enrutamiento Proxy (Outbounds, Reglas, API de Clash) y un Examen final.
* 🚀 **Constructor HAPP / Base64:** Minificación instantánea de código y generación de enlaces profundos `happ://routing/...` para la implementación rápida de reglas de enrutamiento en clientes (v2Ray, Sing-box).
* 🔗 **Compartir e Iframe:** ¿Escribiste una buena configuración? Haz clic en *"Compartir"* y la aplicación generará una URL con tu código cifrado en el interior (¡sin bases de datos!). ¡O copia el código Iframe para integrar el editor en tu sitio!
* 📝 **System Logger:** Pestaña profesional de registros detallados del sistema (precisión de milisegundos) con exportación a `.txt`.
* 🔄 **Actualizaciones OTA Inteligentes:** ¡La aplicación comprueba automáticamente el repositorio oficial de GitHub Releases y te notifica sobre nuevas versiones!
* 🌍 **Multilingüe (i18n):** Cambio instantáneo de la interfaz a 6 idiomas (incluido el soporte RTL para árabe/farsi).

### 📥 Instalación y Lanzamiento

La aplicación no requiere instalar Node.js, servidores ni bases de datos.

1. Descarga `index.html` del último [Release](https://github.com/JINXPIL/json-yaml-ai/releases).
2. Haz doble clic en el archivo para abrirlo en cualquier navegador (Chrome, Edge, Safari).
3. **¡O** simplemente usa la [Versión Web](https://jinxpil.github.io/json-yaml-ai/) directamente desde el repositorio!
