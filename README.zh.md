> [!IMPORTANT]
> **零信任与 100% 隐私。** 该应用程序是一个单一的单体 HTML 文件。所有核心计算、链接生成和解析都在**您的设备内存中严格本地**进行。AI API 密钥仅保存在本地缓存中，并直接发送给提供商（无中间服务器）。

> [!TIP]
> **PWA 和移动端就绪：** 在智能手机（iOS/Android）上打开网站并点击“添加到主屏幕”。该应用程序将作为完全原生的程序安装，甚至可以离线工作（包括离线 AI）！

### 🔥 主要功能（里面有什么？）

* 🌌 **Omni-AI 桥接（在线 AI）：** 连接您的 API 密钥，聊天机器人就会变成天才！内置对 `DeepSeek v4`、`ChatGPT (OpenAI)`、`Claude 3.5 (Anthropic)`、`Gemini` 和免费的 `Qwen/Llama 3` 模型（通过 OpenRouter）的直接支持。神经网络读取您的代码并帮助优化它。
* 🤖 **带有机器学习的离线 AI：** 没有互联网或密钥？没问题！内置的离线机器人将发现语法错误（尾随逗号、单引号）并使用“fix”命令**自动修复它们**。此外，您可以使用 `learn: X = Y` *教* 它新命令。
* 📂 **全局拖放：** 只需将任何 `.json`、`.yaml` 或 `.txt` 文件从桌面直接拖放到浏览器窗口即可立即解析，或将其放入 AI 聊天中进行详细的神经分析。
* 🎨 **游戏引擎 UI（模拟器和调整大小）：** 绝对的界面自由。向任何方向拖动窗口以调整大小。内置的**移动模拟器**允许直接在 PC 上测试移动布局。自定义颜色并一键分享您的主题！
* 🎓 **交互式学院：** 内置 15 步课程：从基本的 JSON 类型到复杂的代理路由配置（Outbounds、Rules、Clash API）和最终考试。
* 🚀 **HAPP / Base64 生成器：** 立即压缩 (Minify) 代码并生成 `happ://routing/...` 深度链接，以便在客户端（v2Ray、Sing-box）中快速实现路由规则。
* 🔗 **分享和 Iframe：** 写了一个很酷的配置？点击 *“分享”*，应用程序会生成一个 URL，您的代码已加密在其中（无数据库！）。或者复制 Iframe 代码将编辑器嵌入到您的网站上！
* 📝 **系统日志：** 专业的详细系统日志选项卡（毫秒精度），支持 `.txt` 导出。
* 🔄 **智能 OTA 更新：** 应用程序自动检查官方 GitHub Releases 仓库并通知您新版本！
* 🌍 **多语言 (i18n)：** 即时将界面切换为 6 种语言（包括对阿拉伯语/波斯语的 RTL 支持）。

### 📥 安装与运行

应用程序无需安装 Node.js、服务器或数据库。

1. 从最新的 [Release](https://github.com/JINXPIL/json-yaml-ai/releases) 下载 `index.html`。
2. 双击文件以在任何浏览器（Chrome、Edge、Safari）中打开。
3. **或者**直接从仓库使用 [Web 版本](https://jinxpil.github.io/json-yaml-ai/)！
