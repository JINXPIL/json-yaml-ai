📖 Энциклопедия

                
1. Введение

                
JSON — строгий стандарт. YAML — для людей.

                
2. Структура JSON

                
Объекты { } и массивы [ ].

                
{"name": "Иван"}
[1, 2, 3]

                
3. Типы данных JSON

                
Только 6 типов.

                
"Строка", 42, 3.14, true, false, null

                
4. Строгие правила

                
Ключи в кавычках.

                
{"name": "Иван"} // Правильно
{name: "Иван"}   // ОШИБКА

                
5. Экранирование

                
Слеши для спецсимволов.

                
\" — двойная кавычка
\\ — слеш
\n — перенос

                
6. Форматирование

                
Minified экономит память.

                
{"a":1,"b":2} // Minified

                
7. YAML: Синтаксис

                
Строгие отступы.

                
server: Proxy
dns:
  - 8.8.8.8

                
8. YAML: Текст

                
Символы | и >.

                
message: |
  Строка 1
  Строка 2

                
9. Что такое Base64 и HAPP?

                
Сжатый формат для Proxy.
