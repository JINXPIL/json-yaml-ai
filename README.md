> [!IMPORTANT]
> **Zero-Trust & 100% Privacy.** Приложение представляет собой единый монолитный HTML-файл. Все базовые вычисления, генерация ссылок и парсинг происходят **строго локально в оперативной памяти вашего устройства**. API-ключи нейросетей сохраняются только в вашем локальном кэше и отправляются напрямую провайдерам (без серверов-посредников).

> [!TIP]
> **PWA & Mobile Ready:** Откройте сайт на смартфоне (iOS/Android) и нажмите «Добавить на главный экран». Приложение установится как полноценная нативная программа, которая будет работать даже без интернета (включая Офлайн-ИИ)!

### 🔥 Главные функции (Что внутри?)

* 🌌 **Omni-AI Bridge (Онлайн ИИ):** Подключите свои API-ключи и чат-бот превратится в гения! Встроенная прямая поддержка `DeepSeek v4`, `ChatGPT (OpenAI)`, `Claude 3.5 (Anthropic)`, `Gemini` и бесплатных моделей `Qwen/Llama 3` (через OpenRouter). Нейросеть читает ваш код и помогает его оптимизировать.
* 🤖 **Офлайн ИИ с машинным обучением:** Нет интернета или ключей? Не проблема! Встроенный офлайн-бот сам найдет синтаксические ошибки (висячие запятые, одинарные кавычки) и **автоматически исправит их** командой "fix". Более того, вы можете *обучить* его новым командам фразой: `запомни: X = Y`.
* 📂 **Global Drag & Drop:** Просто перетащите любой `.json`, `.yaml` или `.txt` файл с рабочего стола прямо в окно браузера для моментального парсинга, или бросьте его в ИИ-чат для детального анализа нейросетью.
* 🎨 **Game Engine UI (Эмулятор & Ресайз):** Абсолютная свобода интерфейса. Тяните окна в любую сторону для изменения размера. Встроенный **Мобильный эмулятор** позволяет тестировать отображение для телефонов прямо на ПК. Настраивайте цвета и делитесь своей темой в 1 клик!
* 🎓 **Интерактивная Академия:** Встроенный курс из 15 шагов: от базовых типов JSON до сложных конфигураций Proxy-маршрутизации (Outbounds, Rules, Clash API) и финального Экзамена.
* 🚀 **Конструктор HAPP / Base64:** Мгновенное сжатие (Minify) вашего кода и генерация `happ://routing/...` диплинков для быстрой имплементации правил маршрутизации в клиенты (v2Ray, Sing-box).
* 🔗 **Share & Iframe:** Нажали *"Поделиться"*? Приложение сгенерирует URL, внутри которого зашифрован ваш код (без баз данных!). Или скопируйте Iframe-код для встраивания редактора на свой сайт!
* 📝 **System Logger:** Профессиональная вкладка детальных системных логов (с точностью до миллисекунд) с выгрузкой в `.txt`.
* 🔄 **Smart OTA Updates:** Приложение само проверяет официальный репозиторий GitHub Releases и уведомляет вас о выходе новых версий!
* 🌍 **Мультиязычность (i18n):** Мгновенное переключение всего интерфейса на 6 языков (включая RTL-поддержку для арабской вязи — Фарси).

### 📥 Установка и Запуск

Приложение не требует установки Node.js, серверов или баз данных. 

1. Скачайте `index.html` из последнего [Релиза](https://github.com/JINXPIL/json-yaml-ai/releases).
2. Кликните по файлу дважды, чтобы открыть его в любом браузере (Chrome, Edge, Safari).
3. **Или** просто пользуйтесь [Web-версией](https://jinxpil.github.io/json-yaml-ai/) напрямую из репозитория!
