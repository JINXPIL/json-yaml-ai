> [!IMPORTANT]
> **Zero-Trust & 100% Privacy.** The app is a single monolithic HTML file. All core calculations, link generation, and parsing happen **strictly locally in your device's RAM**. AI API keys are saved only in your local cache and sent directly to providers (no middleman servers).

> [!TIP]
> **PWA & Mobile Ready:** Open the site on a smartphone (iOS/Android) and tap "Add to Home Screen". The app will install as a fully native program that works even offline (including Offline AI)!

### 🔥 Main Features (What's inside?)

* 🌌 **Omni-AI Bridge (Online AI):** Connect your API keys and the chatbot turns into a genius! Built-in direct support for `DeepSeek v4`, `ChatGPT (OpenAI)`, `Claude 3.5 (Anthropic)`, `Gemini`, and free `Qwen/Llama 3` models (via OpenRouter). The neural network reads your code and helps optimize it.
* 🤖 **Offline AI with Machine Learning:** No internet or keys? No problem! The built-in offline bot will find syntax errors (trailing commas, single quotes) and **automatically fix them** using the "fix" command. Moreover, you can *teach* it new commands with: `learn: X = Y`.
* 📂 **Global Drag & Drop:** Just drag and drop any `.json`, `.yaml`, or `.txt` file from your desktop directly into the browser window for instant parsing, or drop it into the AI chat for detailed neural analysis.
* 🎨 **Game Engine UI (Emulator & Resize):** Absolute interface freedom. Drag windows in any direction to resize. The built-in **Mobile Emulator** allows testing mobile layouts right on your PC. Customize colors and share your theme in 1 click!
* 🎓 **Interactive Academy:** Built-in 15-step course: from basic JSON types to complex Proxy routing configs (Outbounds, Rules, Clash API) and a final Exam.
* 🚀 **HAPP / Base64 Builder:** Instant code Minify and `happ://routing/...` deep link generation for fast routing rule implementation in clients (v2Ray, Sing-box).
* 🔗 **Share & Iframe:** Wrote a cool config? Click *"Share"* and the app generates a URL with your code encrypted inside (no databases!). Or copy the Iframe code to embed the editor on your site!
* 📝 **System Logger:** Professional detailed system logs tab (millisecond precision) with `.txt` export.
* 🔄 **Smart OTA Updates:** The app automatically checks the official GitHub Releases repository and notifies you of new versions!
* 🌍 **Multilingual (i18n):** Instant interface switching to 6 languages (including RTL support for Arabic/Farsi).

### 📥 Installation and Launch

The application does not require Node.js, servers, or databases.

1. Download `index.html` from the latest [Release](https://github.com/JINXPIL/json-yaml-ai/releases).
2. Double-click the file to open it in any browser (Chrome, Edge, Safari).
3. **Or** just use the [Web Version](https://jinxpil.github.io/json-yaml-ai/) directly from the repository!
